Pokédex Web App (Vanilla JavaScript)

A responsive Pokédex application that lets users browse and search through the first 151 Pokémon. The app fetches data from the public PokéAPI, displays Pokémon in a card layout, a nd allows users to view detailed information (stats, types, height, and description) via Bootstrap modals. Features include real-time search, type filtering, loading states, and a fully responsive design using HTML, CSS, JavaScript, and Bootstrap.
